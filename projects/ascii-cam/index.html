Engine

    
—

    
Characters rendered per frame across the viewport grid. Each glyph maps to an average luminance value sampled from the source image.

  

  

    
Process

    
Live video frames are captured at 640×480, divided into a density-scaled character grid. Each cell's RGB channels are averaged, luminance-calculated, then mapped to a 13-level character ramp for output.

  

  

    
Character Ramp

    

      [ . , - ~ : ; = ! * # $ @ ]
    

    
Dark to bright — 13 levels of luminance quantization creating the ASCII aesthetic.
